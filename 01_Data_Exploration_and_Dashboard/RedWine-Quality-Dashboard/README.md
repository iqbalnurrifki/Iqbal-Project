## ğŸ§­ Overview
The project focuses on **Exploratory Data Analysis (EDA)** to understand the physicochemical factors that influence wine quality.  
It provides a set of interactive tools for visualizing variable distributions, correlations, and relationships between chemical properties and the final wine quality scores.

The dataset used is the **Red Wine Quality dataset** from the UCI Machine Learning Repository.
